## How to Create a Pull Request (Step-by-Step)

### Step 1: Fork the Repository

Click "Fork" at the top right.

### Step 2: Clone Your Fork

```bash
git clone https://github.com/PonnamSriSai/tds_project_1.git
cd tds_project_1
```

### Step 3: Create a Branch

```bash
git checkout -b add-my-pr
```

### Step 4: Make a Small Change
- Open the file:
```bash
/contributors/contributors.md
```

- Add your profile details at the bottom in the following format:

## Your Name
- GitHub: https://github.com/your-username
- IITM Email: your-email@ds.study.iitm.ac.in
- Short Bio: 1–2 line introduction

### Step 5: Commit

```bash
git add .
git commit -m "Add profile - Your Full Name"
```

### Step 6: Push

```bash
git push origin add-my-pr
```

### Step 7: Open a Pull Request (Detailed Guide)

After pushing your branch, follow these steps carefully:

---

#### 1️⃣ Go to Your Fork on GitHub

- Open GitHub.
- Click your profile picture (top right).
- Click **Your repositories**.
- Open your fork of this repository.

⚠️ Make sure you are inside **your fork**, not the original repository.

---

#### 2️⃣ Click “Compare & Pull Request”

If you just pushed your branch, GitHub will show a yellow notification box:

> “Your branch was recently pushed.”

Click the **Compare & pull request** button.

If you do NOT see that button:

1. Click the **Pull requests** tab.
2. Click **New pull request**.
3. Make sure the branches are set correctly:
   - **Base repository** → original repository
   - **Base branch** → `main`
   - **Head repository** → your fork
   - **Compare branch** → `add-my-pr`
4. Click **Create pull request**.

---

#### 3️⃣ Write the Pull Request Title

Use this exact format:

Add profile - Your Full Name

#### 4️⃣ Write the Pull Request Description

Copy and paste this template:

Added my profile to contributors.md as per repository guidelines.

---

#### 5️⃣ Double-Check Before Submitting

Before clicking **Create pull request**, confirm:

- ✅ Only `contributors.md` is changed  
- ✅ No other files are modified  
- ✅ Your name is added correctly  
- ✅ Formatting is not broken  
- ✅ Email is correct  

If anything else is changed, go back and fix it before submitting.

---

#### 6️⃣ Click “Create Pull Request”

Click the green **Create pull request** button.

Your PR is now submitted for review.

---

#### 7️⃣ After Submitting

- Wait for review.
- If changes are requested, update your branch and push again.
- Do NOT open multiple PRs.
- Do NOT close and reopen repeatedly.

One clean PR = fast merge.
