# Code of Conduct

## Purpose

This repository is created for TDS students to practice responsible open-source contribution.

To maintain a respectful and professional environment, all contributors are expected to follow this Code of Conduct.

---

## Our Standards

All participants must:

- Be respectful and professional in communication
- Use clear and appropriate language
- Accept feedback gracefully
- Follow repository contribution guidelines
- Respect maintainer decisions

---

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment or discrimination of any kind
- Offensive or abusive language
- Spamming pull requests or issues
- Submitting intentionally misleading information
- Editing or deleting other contributors' entries
- Creating conflicts intentionally
- Impersonating another student

Pull requests violating these standards will be closed immediately.

Severe violations may result in being blocked from the repository.

---

## Pull Request Etiquette

When submitting a PR:

- Keep it minimal and focused
- Follow formatting instructions exactly
- Do not argue aggressively in reviews
- If changes are requested, respond politely and update your branch
- Do not open multiple PRs for the same contribution

Maintainer time is limited. Respect it.

---

## Reporting Issues

If you encounter:

- Misconduct
- Fake profiles
- Repeated rule violations

Please open an issue describing the concern clearly and respectfully.

---

## Maintainer Responsibility

Maintainers will:

- Review pull requests fairly
- Apply rules consistently
- Communicate clearly
- Merge valid contributions promptly

Maintainers reserve the right to close PRs that do not follow guidelines.

---

## Academic Integrity Notice

This repository is part of an academic exercise.

All contributors are responsible for:

- Using their real IITM email
- Submitting only their own profile
- Following assignment requirements honestly

Academic misconduct is the responsibility of the individual contributor.

---

By contributing to this repository, you agree to follow this Code of Conduct.
